NumpadA (prototype)
=====

A fully programmable standard numpad using Cherry MX switches.

Keyboard Maintainer: Inexpensive Computers
Hardware Supported: NumpadA PCB, QMK Proton C
Hardware Availability: [Inexpensive Computers](https://boards.inexpensivecomputers.net)

Make example for this keyboard (after setting up your build environment):

    make handwired/inexcomp/numpada:default

See [build environment setup](https://docs.qmk.fm/#/getting_started_build_tools) then the [make instructions](https://docs.qmk.fm/#/getting_started_make_guide) for more information.
